# VK Quiz App

Веб-приложение для проведения онлайн-квизов в режиме реального времени.

## Описание проекта

VK Quiz App — это MVP-прототип платформы для проведения квизов.  
Организатор может создать комнату и запустить квиз, а участники — подключиться по коду комнаты и проходить задания синхронно в режиме реального времени.

Проект разработан в рамках проектной задачи VK.

---

## Реализованная функциональность

### Пользовательская часть
- Главная страница приложения
- Форма регистрации
- Форма авторизации
- Подключение к комнате по коду
- Прохождение квиза
- Отображение вопросов и вариантов ответов
- Лидерборд участников

### Организатор
- Панель организатора
- Создание комнаты
- Запуск квиза
- Переключение вопросов
- Просмотр списка игроков и баллов

### Real-time функциональность
- Синхронное прохождение квиза
- Обновление вопросов в реальном времени
- Автоматическое обновление лидерборда
- Передача данных через WebSocket (Socket.IO)

## Установка и запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/Fasko1/vk-quiz-app.git
```

---

## Запуск frontend

```bash
cd client
npm install
npm run dev
```

Frontend запускается по адресу:

```bash
http://localhost:5173
```

---

## Запуск backend

```bash
cd server
npm install
npm run dev
```

Backend запускается по адресу:

```bash
http://localhost:5000
```

---

## Реализованный сценарий работы

1. Организатор открывает панель администратора
2. Создаётся комната с кодом
3. Участники подключаются по коду комнаты
4. Организатор запускает квиз
5. Вопросы отображаются у всех участников одновременно
6. Игроки отвечают на вопросы
7. Баллы обновляются автоматически
8. По окончании показывается лидерборд
