# VK Quiz App

Веб-приложение для проведения онлайн-квизов в реальном времени.

## Функциональность

- Регистрация пользователей
- Авторизация
- Создание квизов
- Прохождение квиза
- Отображение вопросов и вариантов ответов
- Получение данных с backend API

## Технологии

Frontend:
- React
- React Router
- CSS

Backend:
- Node.js
- Express
- REST API

## Запуск проекта

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
